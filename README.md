## Features
- 3x3 Grid Interface: A responsive board that scales for mobile and desktop.
- Turn-Based Logic: Alternating inputs between "X" and "O" players.
- Win Detection: Algorithms to check for three matching symbols horizontally, vertically, or diagonally.
- Draw/Tie State: Logic to detect when the board is full with no winner.
- Reset Functionality: A "Play Again" button to clear the board and reset the game state without refreshing the page.
